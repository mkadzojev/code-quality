This package is a backend quality control package

## Installation
This package requires PHP 8.1 and Laravel 10.0 or higher.

### require package
```
composer require mikdev/backend-quality --dev
```

### install configs
```php
php artisan quality:install
```

## Testing
```bash
composer test
```
